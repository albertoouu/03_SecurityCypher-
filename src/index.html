Descifrar

Key:



Instrucciones:
1. Introduzca dentro del recuadro blanco "Key" la llave para descifrar.
2. Escriba dentro del recuadro grande el texto a Descifrar (solo se admiten letras, mayúsculas y minúsculas sin acentos).
3. Presionar "CIFRAR".
4. Presione "REGRESAR" para volver a la pantalla de Cifrado / Descifrado.
